    Project 1

    Security Alarm System

    This project focuses on the detection of unauthorized entry using ultrasonic sensors. It employs an Arduino Uno board to control the system, while mobile phone notifications are sent as alerts in case of a breach. The system is designed to provide real-time security updates, ensuring prompt action when unauthorized access is detected.

    Smart Car Parking System

    Developed as part of a hackathon, this project features a website that enables users to book parking slots in advance, pay before arrival, and manage bookings. The system includes functionalities like imposing fines for non-updated departure times and extra charges for last-minute cancellations, ensuring better management of parking spaces.
